## Overview:

- Jump is a swift application that aims to eliminate mouse movement.
- Jump is an application that runs in the terminal

## Activation

- Jump activates when the following hotkey is pressed: (cmd + ctrl + shift + opt + [space])
- Upon activation the user is presented with a text box
- Keyboard focus is shifted to that text box

## Action

- The user types text into the text box
- The application will search the focsed window for elements or components or accessibility labels that match the text
- The matching elements will be highlighted in a transparent green color
- When one element is uniquely identified the user may press the Enter key to move the mouse to that location
- The text box is removed after pressing enter
